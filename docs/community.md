# 커뮤니티

A2UI 커뮤니티에 오신 것을 환영합니다! 우리는 에이전트 주도 인터페이스의 미래를 함께 구축하고 있습니다.

## 참여하기

A2UI는 Apache 2.0 라이선스 하에 공개된 오픈 소스 프로젝트입니다. 개발자, 연구자, 그리고 에이전트 기반 사용자 인터페이스(Agentic UI) 발전에 관심 있는 모든 분의 기여를 환영합니다.

## 커뮤니티 쇼케이스

!!! info "곧 출시 예정..."
    커뮤니티 프로젝트, 예시, 테마, 렌더러, 커스텀 컴포넌트 등을 가장 잘 보여줄 방법을 고민 중입니다. 4분 이내의 데모 영상과 GitHub Discussions에 링크된 코드 샘플은 여러분의 작업물을 뽐낼 아주 좋은 방법입니다.

## 프로젝트 파트너

A2UI는 여러 조직과의 협업을 통해 개발되고 있습니다:

### Google Opal

[Opal](http://opal.google)은 사용자가 자연어를 사용하여 AI 미니 앱을 구축하고, 편집하고, 공유할 수 있게 해줍니다. Opal 팀은 A2UI 개발의 핵심 기여자였습니다.

### Flutter

[Flutter용 GenUI SDK](https://docs.flutter.dev/ai/genui)는 Flutter 애플리케이션에서 동적이고 개인화된 UI를 생성하기 위해 A2UI를 UI 선언 형식으로 사용합니다.

### Gemini Enterprise

A2UI는 [Gemini Enterprise](https://cloud.google.com/gemini-enterprise?e=48754805)에 통합되어 엔터프라이즈 애플리케이션 내에서 커스텀 에이전트가 풍부하고 대화형인 UI를 렌더링할 수 있도록 지원하고 있습니다.

### AG UI / CopilotKit

[AG UI](https://ag-ui.com/)와 [CopilotKit](https://www.copilotkit.ai/)은 A2UI와의 즉각적인 호환성을 제공하여, 개발자들이 에이전트로부터 동적 UI를 렌더링하고 상태가 동기화된 풍부한 애플리케이션을 빌드할 수 있게 합니다.

### A2A

Google의 [A2A 팀](https://a2a-protocol.org/)은 A2A 기술 운영 위원회(TSC)의 지원과 함께 A2UI 개발의 핵심 기여자로 참여해 왔습니다.

### ... 그 외 다수

많은 조직과 개인이 A2UI의 발전에 기여하고 있습니다.

A2UI에 중대한 기여를 하셨다면, 이 목록에 이름을 올려주세요.

## 행동 강령

우리는 모두에게 환영받고 포용적인 환경을 제공하기 위해 최선을 다하고 있습니다. 모든 참가자는 다음 사항을 준수해야 합니다:

- 존중하고 배려하는 태도 갖추기
- 초보자를 환영하고 시작을 돕기
- 커뮤니티에 가장 도움이 되는 것에 집중하기
- 다른 사람에게 공감하기

부적절한 행동은 용납되지 않습니다. 우려되는 사항은 프로젝트 유지 관리자에게 보고해 주세요.

## 기여에 대한 인정

모든 기여에 감사드립니다! 기여자는 다음 항목에서 인정받게 됩니다:

- GitHub 기여자 그래프
- 릴리스 노트
- 프로젝트 감사의 글

중대한 기여자는 프로젝트의 유지 관리자(Maintainer) 팀에 초대될 수 있습니다.
